Чек-лист для калькуляторa. Консольный калькулятор получает из командной строки два числа и операцию, после
чего выполняет операцию и отображает в консоли результат.

1. Проверить порядок аргументов
	А) число,  число, оперция
	B) число, операция, число
	С) опериция, число, число
2. Проверить опериции *, +, -, % с целочисленными аргументами
3. Проверить % на 0
	А) число 0 %
4. Проверить работу с отрицательными аргументами
	А) -число, -число, +
	B) -число, -число, -
	С) число, -число, *; (-число, число, *)
	Д) число, -число, %; (число, -число, %)
5. Проверить работу с вещественными числами
6. Если один из аргументов недопустимый символ (не число, не операция)
7. Пробелы до и после аргументов
